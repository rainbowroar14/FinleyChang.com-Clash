Cards use tournament-level 9 stats from the Clash Royale wiki (hit speed, first hit, HP, damage) scaled into this arena — Mini P.E.K.K.A (4), Knight (3), Skeletons (3).

Play

Training is offline vs AI. Battle joins a live queue; when two players are waiting, you are paired and a match id is written to Firestore (shared troop sync can build on that).

About Firebase: it is not only a file dump. This project uses Cloud Firestore — a document database that syncs in real time — for the queue and match records. The same Firebase project can later add Auth, Cloud Functions, etc.
